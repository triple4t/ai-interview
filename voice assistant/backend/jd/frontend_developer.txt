JOB DESCRIPTION: FRONTEND DEVELOPER

Position: Frontend Developer
Department: Engineering
Location: Remote/Hybrid
Employment Type: Full-time
Experience Level: Mid to Senior (3-7 years)

COMPANY OVERVIEW:
We are building an innovative AI-powered interview assistant platform that revolutionizes the hiring process through intelligent voice interactions, real-time transcription, and comprehensive candidate evaluation.

JOB SUMMARY:
We are seeking a talented Frontend Developer to join our engineering team. You will be responsible for creating intuitive, responsive, and high-performance user interfaces for our AI interview platform. You'll work with modern frontend technologies including React, TypeScript, and real-time communication features.

KEY RESPONSIBILITIES:

1. User Interface Development:
   - Build responsive and accessible user interfaces using React/Next.js
   - Implement modern UI/UX designs with attention to detail
   - Create reusable component libraries and design systems
   - Ensure cross-browser compatibility and mobile responsiveness

2. Real-time Communication Features:
   - Implement WebRTC-based voice/video communication interfaces
   - Create real-time chat and messaging components
   - Handle live transcription display and synchronization
   - Manage audio/video device selection and controls

3. State Management & Data Flow:
   - Implement efficient state management using Redux, Zustand, or Context API
   - Handle complex data flows and API integrations
   - Manage real-time data updates and synchronization
   - Implement optimistic updates and error handling

4. Performance Optimization:
   - Optimize application performance and loading times
   - Implement code splitting and lazy loading strategies
   - Optimize bundle sizes and reduce unnecessary re-renders
   - Monitor and improve Core Web Vitals

5. Authentication & User Experience:
   - Implement user authentication flows and protected routes
   - Create seamless onboarding and user registration experiences
   - Handle form validation and error messaging
   - Implement progressive web app features

6. Interview Platform Features:
   - Build interview scheduling and management interfaces
   - Create candidate evaluation and scoring displays
   - Implement real-time interview progress tracking
   - Design dashboard and analytics visualizations

7. Testing & Quality Assurance:
   - Write comprehensive unit and integration tests
   - Implement end-to-end testing with Cypress or Playwright
   - Perform cross-browser and device testing
   - Conduct accessibility testing and compliance

8. Collaboration & Code Quality:
   - Work closely with designers and backend developers
   - Participate in code reviews and maintain code quality
   - Document components and APIs
   - Mentor junior developers and share knowledge

Development Tools:

- Performance monitoring and optimization tools
- Design tools (Figma, Sketch) collaboration

Soft Skills:

- Excellent communication and collaboration skills
- Attention to detail and user experience focus
